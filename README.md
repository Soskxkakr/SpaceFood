## SpaceFood - Website of Food Ordering System created in ASP.NET
### Features of SpaceFood
1.  New User Registration
2.  View Food
3.  Add Food to Cart
4.  View Cart
5.  Delete a food from Cart
6.  Checkout
7.  View Profile
8.  View Order History
9.  Admin Controls (CRUD for Food and Users)
